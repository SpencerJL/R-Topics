# R markdown 简介

>  `R Markdown`是通过R来书写可重复动态报告的 ⼀种格式。利⽤它在幻灯⽚、pdf、
html⽂档、Word⽂件中嵌⼊R代码和结果。


## 文件列表


1. `main.RMD` 和 `main.pdf`是报告的内容。

2. `img`文件夹是报告需要的图片

3. `example`文件夹内存放的各个部分的示例。
