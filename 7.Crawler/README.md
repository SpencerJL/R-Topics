# 网络爬虫技术简介

>  网络爬虫,又被称为网页蜘蛛,网络机器人,它是指按照一定的规则,自动抓取万维网(World Wide Web)信息的程序或脚本。


## 文件列表


1. `Presentation`文件夹内存放的是报告的内容。

2. `HTML`文件夹内存放的是 HTML 示例。

3. `Code`文件夹内存放的完整的爬虫代码。
